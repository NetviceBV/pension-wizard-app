

## Update Favicon

Replace the current favicon reference in `index.html` with the two Prikkl favicon links provided by the user.

### Changes in `index.html`
- Remove the existing `<link rel="icon" href="/favicon.ico" type="image/x-icon" />` line
- Add the two provided favicon links:
  ```html
  <link rel="icon" type="image/png" sizes="16x16" href="https://prikkl.nl/wp-content/themes/prikkl/dist/img/favicon/favicon-16x16.png">
  <link rel="icon" type="image/png" sizes="32x32" href="https://prikkl.nl/wp-content/themes/prikkl/dist/img/favicon/favicon-32x32.png">
  ```
